# React Stack

This is a starter template for a React app with the following technologies:

- React
- React Router
- Tailwind CSS
- Clerk
- Vite
- Typescript
- ShadCN/UI

### Environment variables

```sh
VITE_CLERK_PUBLISHABLE_KEY=
CLERK_SECRET_KEY=
CLERK_SIGN_IN_FALLBACK_URL=
CLERK_SIGN_IN_URL=
CLERK_MODE=
```

### Commands

```
bun run dev
```
